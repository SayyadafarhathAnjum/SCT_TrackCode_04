# 🖐️ Hand Gesture Recognition Model

A deep learning model to identify and classify 
different hand gestures from image data, enabling 
intuitive human-computer interaction.

## 📌 Task
SkillCraft Technology - Task 04

## 📊 Dataset
[Hand Gesture Dataset - Kaggle](https://www.kaggle.com/datasets/gti-upm/leapgestrecog)

## 🔧 Tech Stack
- Python
- TensorFlow / Keras (CNN Model)
- OpenCV
- NumPy, Matplotlib
- Streamlit (Live App)

## 🚀 How to Run
pip install -r requirements.txt
python gesture_model.py
